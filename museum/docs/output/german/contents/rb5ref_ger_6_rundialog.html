RUNDIALOG
 
Syntax: RUNDIALOG Event-String = Timeout
Beschreibung: Wartet auf ein Ereignis mit oder auf einem RapidBATCH-eigenen Dialog der aktuellen Script-Instanz.

Als Timeout wird eine Timeout-Wartezeit in Millisekunden angegeben. Diese ist optional, bei Übergabe des Timeout-Wertes '0' findet kein Timeout statt (d.h. die Funktion wartet so lange, bis ein vom Benutzer erzeugtes Event ausgelöst wird).
Rückgabewert der Funktion ist, falls ein Timeout eintrat, ein Leerstring, ansonsten ein so genannter Event-String, der im folgenden Format generiert wird:

Ereignis_Widget-Label

Wobei "Ereignis" einer der folgend aufgeführten Event-Identifiers (Ereignis-Bezeichner) sein kann und "Widget-Label" für das bei NEWDIALOG definierte Label des Widgets steht.

Ereignis-Bezeichner: Beschreibung:
click Ein Widget wurde angeklickt
dblclick Ein Widget wurde doppelt angeklickt
change Ein Widget wurde geändert (z.B. Eingabefelder)
close Schlieﬂen-Button eines Dialogs wurde geklickt
size Dialog wurde skaliert
move Dialog wurde bewegt
mousemove Maus wurde über einem Dialog bewegt
ok Die Taste "Enter" wurde auf dem Dialogfenster gedrückt
esc Die Taste "Esc" wurde auf dem Dialogfenster gedrückt

Es werden Ereignisse von allen angezeigten Dialogfenstern, die das RapidBATCH-Script bereitstellt, angenommen und zurückgegeben. Anhand des Event-Strings können dann entsprechende Aktionen zum eingetretenen Ereignis durchgeführt werden.
Beispiele: rem Dialog erzeugen
newdialog 'Fenster', 'dialog', '1|1|200|100'
newdialog 'Fenster:Button', 'button', '2|2|192|72'

rem Button beschriften
letdialog 'Fenster:Button', 'caption', 'Hello World'

rem Fenster anzeigen
letdialog 'Fenster', 'visible', [true]

repeat
    rem Dialog ausführen (kein Timeout!)
    rundialog [ereignis] = '0'

    rem Ereignis-String auswerten
    if [ereignis] = 'click_Fenster:Button'
        echo 'Der Button wurde angeklickt.'
    endif

    rem Wird der Schlieﬂen-Button des Fensters geklickt,
    rem muss ebenfalls eine entsprechende Prüfung statt-
    rem finden
until [ereignis] = 'close_Fenster'

end
Bemerkungen: Ob einzelne Widgets ein Ereignis auslösen sollen oder nicht kann mit der LETDIALOG-Anweisung bestimmt werden.
