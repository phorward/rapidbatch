[errorcode]
 
Beschreibung: Die Variable [errorcode] ist eine von vielen Anweisungen und Funktionen benutzte Statusvariable, die Auskunft darüber gibt, ob eine Aktion erfolgreich war oder nicht oder welchen Status eine Aktion angenommen hat (z.B. welcher Button in einer MSGBOX-Meldung oder CONFIRM-Meldung gedrückt wurde).

In der Befehlsreferenz finden Sie im Bereich "Bemerkungen" jeweils Auskunft, welche Fehlerwerte [errorcode] von der jeweiligen Anweisung/Funktion zugewiesen werden können.
