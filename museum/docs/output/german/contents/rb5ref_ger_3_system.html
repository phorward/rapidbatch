SYSTEM
 
Syntax: SYSTEM Kommando
Beschreibung: Führt ein Betriebssystemkommando oder Programm aus. Dies sind unter Windows reine DOS-Befehle. Das Script wartet solange mit der Ausführung bis das mit SYSTEM gestartete Programm beendet wurde.
Beispiele: system 'dir >list.txt'
Bemerkungen: Die Variable [errorcode] erhält den Wert -1, wenn kein COMMAND.COM oder CMD.EXE gefunden wurde, andernfalls 0.
