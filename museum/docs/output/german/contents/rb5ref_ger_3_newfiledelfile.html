NEWFILE, DELFILE
 
Syntax: NEWFILE Dateiname
DELFILE Dateiname
Beschreibung: NEWFILE legt eine neue, leere Datei an. Existiert die Datei bereits, wird Sie mit der neuen, leeren Datei überschrieben.
DELFILE löscht eine vorhandene Datei unwiderruflich.
Beispiele: rem Neue Datei anlegen
newfile 'neu.txt'

rem Temp-Datei löschen
delfile 'ausgabe.tmp'
Bemerkungen: Ist die jeweilige Operation erfolgreich, so erhält [errorcode] den Wert 0, ansonsten -1.
