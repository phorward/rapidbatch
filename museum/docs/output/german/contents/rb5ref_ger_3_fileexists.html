FILEEXISTS
 
Syntax: FILEEXISTS Existenz = Dateiname
Beschreibung: Prüft die Existenz der angegebenen Datei. Wurde die Datei gefunden, gibt die Funktion den Wert 0, ansonsten -1 zurück.
Beispiele: fileexists [errorcode] = 'c:\windows\calc.exe'
if [errorcode] = '0' echo 'Die Datei CALC.EXE existiert!'
Bemerkungen: keine
