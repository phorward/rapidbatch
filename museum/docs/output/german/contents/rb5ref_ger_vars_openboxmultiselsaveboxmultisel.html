[OpenBox_MultiSel], [SaveBox_MultiSel]
 
Beschreibung: Wenn [OpenBox_MultiSel] bzw. [SaveBox_MultiSel] auf den Wert 0 gesetzt wird, lassen sich in den Dialogen OPENBOX bzw. SAVEBOX mehr als eine Datei selektieren.

Die Rückgabe der Funktionen ist dann wie folgt: "Pfadvorsatz|Datei1|Datei2|Datei3 ..." wobei anstatt Pipe (|) das in [std_sep] definierte Standard-Separationszeichen eingesetzt wird.
