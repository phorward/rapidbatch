TRIMVAR
 
Syntax: TRIMVAR Ergebnis = Ausdruck
Beschreibung: Entfernt führende und nachfolgende Leerzeichen aus einer Zeichenfolge und gibt diese zurück.
Beispiele: [text] = '    RapidBATCH  '
echo '***' # [text] # '***'
trimvar [trim_text] = [text]
echo '***' # [trim_text] # '***'
end
