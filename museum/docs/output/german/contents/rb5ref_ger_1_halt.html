HALT
 
Syntax: HALT
Beschreibung: Beendet das Script sofort und gibt alle verwendeten Speicher- und Systemressourcen frei.
Beispiele: chdir 'C:\Test'
if [errorcode] = '-1'
    echo 'C:\Test existiert nicht!'
    rem Script beenden
    halt
endif
Bemerkungen: keine
