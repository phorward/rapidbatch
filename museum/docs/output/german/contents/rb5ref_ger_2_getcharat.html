GETCHARAT
 
Syntax: GETCHARAT Zeichen = Stringausdruck, Zeichenposition
Beschreibung: Liefert das Zeichen an der gegebenen Position im gegebenen Stringausdruck zurück.
Beispiele: [text] = 'Hello World'
rem       12345678901
getcharat [chr] = [text], '5'
echo 'Das Zeichen an Position 5 im String ' # [text] # ' ist ein "' # [chr] # '"'
end
