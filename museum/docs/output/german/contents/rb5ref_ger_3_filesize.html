FILESIZE
 
Syntax: FILESIZE Dateigröße = Dateiname
Beschreibung: Ermittelt die Größe einer Datei und gibt diese Größe in Bytes zurück.
Beispiele: filesize [bytes] = 'c:\command.com'
echo 'COMMAND.COM hat eine Größe von ' # [bytes] # ' Bytes'
end
Bemerkungen: Um eine Dateigröße in Kilo-, Mega-, Giga- oder Terabyte auszugeben muss der Wert jeweils durch 1024 geteilt werden. Für Kilobyte teilt man 1-mal durch 1024, für Terabyte 4-mal durch 1024, usw.
