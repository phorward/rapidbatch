COPYFILE, RENAMEFILE
 
Syntax: COPYFILE Quelldatei, Zieldatei
RENAMEFILE Quelldatei, Zieldatei
Beschreibung: COPYFILE kopiert eine Quelldatei in eine Zieldatei.
RENAMEFILE verschiebt oder benennt eine Quelldatei in eine Zieldatei um.
Beispiele: rem Datei kopieren...
copyfile 'c:\jmksf\rb5\samples\hello.rb', 'a:\hello.rb'

rem Datei umbenennen...
renamefile 'c:\test.neu', 'c:\test.old'
Bemerkungen: Ist die jeweilige Operation erfolgreich, so erhält [errorcode] den Wert 0, ansonsten -1.
