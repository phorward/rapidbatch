GETLEN
 
Syntax: GETLEN Länge = Ausdruck
Beschreibung: Gibt die Anzahl der Zeichen (= Länge) des übergebenen Stringausdrucks zurück.
Beispiele: getlen [len] = 'Hello World'
echo 'Der Text "Hello World" ist ' # [len] # ' Zeichen lang.'
end
