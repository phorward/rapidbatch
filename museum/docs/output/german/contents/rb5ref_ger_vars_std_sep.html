[std_sep]
 
Beschreibung: Die Variable [std_sep] enthält den Standard-Listenseparator.
Dies ist ein Zeichen, welches bei Listen, die z.B. bei LISTBOX oder OPENBOX übergeben werden, als Trennzeichen für die einzelnen Elemente verwendet wird.

[std_sep] enthält defaultmäßig ein Pipe-Zeichen (|). Es kann immer nur ein einzelnes Zeichen als Separator definiert werden, wenn ein String mit mehreren Zeichen an [std_sep] übergeben wird, wird nur das erste Zeichen als Separator verwendet.
