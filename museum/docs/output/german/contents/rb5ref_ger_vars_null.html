[null]
 
Beschreibung: Die Pseudo-Variable [null] wird eingesetzt, um beispielsweise einen nicht benötigten Parameter, der an eine Prozedur übergeben wird, anzugeben oder um den Rückgabewert einer Funktion, der nicht benötigt wird (was manchmal der Fall sein kann) abzufangen. [null] ist in dem Sinne keine Variable. Werte, die an [null] übergeben werden, werden im Speicher sofort verworfen; Wird [null] ausgelesen, enthält sie einen Leerstring.
