[case_sensitivity]
 
Description: If [case_sensitivity] is set to '-1' (or [false]), functions like GETTOK, GETPOS, REPLACEVAR and CNTVAR do not differ between upper- and lower-case order in the strings to be searched for. For example in the case of REPLACEVAR, when the string "Hello" should be replaced, the function also replaces the word "HELLO", "hello" and "HeLlO" if when they are contained in the string to be seached in. Else, only the word "Hello" with a beginning, upper-case letter "H" and a lower-case "ello" will be replaced.
