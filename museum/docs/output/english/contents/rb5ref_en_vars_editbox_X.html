[EditBox_Ok], [EditBox_Cancel], [EditBox_Height], [EditBox_Width], [EditBox_X], [EditBox_Y]
 
Description: Using the variables [EditBox_Ok] and [EditBox_Cancel], the captions of the OK- and Cancel-buttons displayed on the build-in EDITBOX-dialog can be changed and modified.

By setting the variables [EditBox_X] and [EditBox_Y] to any desired pixel value, the dialog's default position on the particular axis can be set and modified. By setting the particular variable to a value of 0, the dialog is automatically centered on the particular axis. This is the default setting.

The variables [EditBox_Width] and [EditBox_Height] let you change and modify the dialog's default size.
