GETFILETIME, LETFILETIME
 
Syntax: GETFILETIME time-of-last-modification = filename
LETFILETIME filename, time-of-last-modification
Description: GETFILETIME reads the last-modified-time of a file and returns this in the format "hhmmss".
LETFILETIME resets the last-modified-time of a file to any desired time value, using the same format returned by GETFILETIME.
Examples: newfile 'test.txt'
getfiletime [t] = 'test.txt'

echo 'test.txt was modified at ' # [t]

rem So let's reset the file's modification time to 2:45 pm
letfiletime 'test.txt', '144500'

getfiletime [t] = 'test.txt'
echo 'test.txt was "modified" at ' # [t]
delfile 'test.txt'
Remarks: If the file was not found, [errorcode] gets from LETFILETIME as well as GETFILETIME the value '-1', else '0'. The returned value of GETFILETIME is in this case an empty string.

Use GETFILEDATE and LETFILEDATE to read and manipulate the modification date of a file.
