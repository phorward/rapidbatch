GETCHARAT
 
Syntax: GETCHARAT character = expression, positions
Description: Returns the character at the specified position in the string-expression.
Examples: [text] = 'Hello World'
rem       12345678901
getcharat [chr] = [text], '5'
echo 'The character at position 5 in the string "' # [text] # '" is "' # [chr] # '"'
end
Remarks: none
