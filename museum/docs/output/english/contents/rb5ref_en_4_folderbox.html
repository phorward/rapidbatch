FOLDERBOX
 
Syntax: FOLDERBOX directory = prompting-text, pre-selected-path
Description: The FOLDERBOX-function provides a standard folder selection dialog to select a folder from the file system.

As parameter, this function expects a prompting-text and an optional, pre-selected path that is directly opened by the dialog. The second parameter can left blank or specified as [null], so no path is pre-selected.
Examples: folderbox [path] = 'Please select a folder:', 'C:\jmksf\rb5'

if [path] ! ''
    echo 'The folder you selected is: ' # [path]
else
    echo 'No folder selected.'
endif
Remarks: Use the dialogs OPENBOX and SAVEBOX to provide a file selection possibility. FOLDERBOX is a dialog provided by the system, so neither its position can be changed nor its button-captions. These captions are automatically displayed in the particular language of the machine it is run with.
