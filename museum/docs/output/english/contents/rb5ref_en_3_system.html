SYSTEM
 
Syntax: SYSTEM command
Description: Executes an operating system command (DOS-commands). The script cannot continue before the SYSTEM-instruction finished its command.
Examples: system 'dir >list.txt'
Remarks: The variable [errorcode] gets the value '-1', if no COMMAND.COM (Win 95, 98, ME) or no CMD.EXE (Win NT, 2000, XP, 2003, Vista) was found, else '0'.
