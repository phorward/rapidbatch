DISKFREE, DISKSIZE
 
Syntax: DISKFREE disk-space = drive-letter
DISKSIZE disk-space = drive-letter
Description: DISKFREE returns the rest of free space on a drive, in bytes.
DISKSIZE returns the whole capacity a drive provides, in bytes.
Examples: diskfree [df] = 'C:\'
echo 'There are already ' # [df] # ' bytes free on drive C:'
diskfree [ds] = 'C:\'
echo 'Drive C: has a capacity of ' # [ds] # ' bytes.'
end
Remarks: To convert a file size to kilo-, mega-, giga- or terrabyte, the byte-value must be divided with 1024 in each case.
For example, a MP3-file has a size of 3853881 bytes. 3853881 divided by 1024 are 3763.55 kilobytes. 3763.55 kilobytes divided again by 1024 are 3.67 mega bytes. This can be continued up to terrabyte (1 terrabyte is 1024 gigabytes).
