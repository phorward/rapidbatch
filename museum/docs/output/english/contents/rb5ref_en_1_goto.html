GOTO
 
Syntax: GOTO label
.
.
.
%label
Description: GOTO calls a specified label in the script, and continues the script execution behind this label. This possibility allows you to write ultra-complex scripts. A label is specified in the script with a beginning %-character.

Since RapidBATCH 5.0, it is possible to call variable labels, so that also a single variable, containing the target-label or a target-label composed of both variable and static parts can be specified.
Examples: rem performing a static label call
echo 'Welcome to the GOTO test script!'
goto 'next'

echo 'You will never see this message :('

%next

rem performing a dynamic label call
include 'dialog.rb'
btnmenu [speed] = 'Please select a speed:', '20 mph|60 mph|100 mph'

goto [speed]

%20 mph
echo 'This speed is very slow.'
goto 'continue'

%60 mph
echo 'This speed is well!'
goto 'continue'

%100 mph
echo 'TOO FAST! You wanna race???'

%continue
echo 'Enjoy your ' # [speed] # ' ;) ...'
Remarks: The label specified at GOTO may not begin with the %-character.
When using GOTO within a procedure, it is not allowed to call a label out of the current or in an other procedure. In this case, the script execution stops with a corresponding error message. Calling labels that are defined in included files is possible.
