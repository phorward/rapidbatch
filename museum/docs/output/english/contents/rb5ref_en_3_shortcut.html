SHORTCUT
 
Syntax: SHORTCUT shortcut-file, shortcut-path, parameters, working-directory, icon, show-mode
Description: Creates a shortcut.

The shortcut is saved under the as shortcut-file specified path, at which the .LNK-file extension for shortcut files is automatically appended by the instruction. shortcut-path defines the path to the executable or document the shortcut is associtated with. parameter, working-directory and icon are optional parameters, and can be left blank if they are not required. As show-mode-parameter, the values 'SHOW', 'MINIMIZED' or 'MAXIMIZED' can be used. The 'HIDE'-mode is not possible when specified to shortcuts.
Examples: [userdir] = 'C:\Documents and Settings\' # [curuser] # '\Desktop'
[prog] = 'C:\jmksf\rb5\rbb4.exe'

shortcut [userdir] # '\RapidBATCH Schortcut', [prog], '', '', '', 'show'

if [errorcode] = [true]
    echo 'The shortcut has been created successfully!'
else
    echo 'Error when creating shortcut.'
endif
Remarks: [errorcode] gets the value '-1' in case of an error, else '0'.
