UPVAR, LOWVAR
 
Syntax: UPVAR result = expression
LOWVAR result = expression
Description: UPVAR converts all characters of a string-expression to upper-case order.
LOWVAR converts all characters of a string-expression to lower-case order.

Characters, which are already in upper/lower-case order are simply ignored by both functions.
Examples: [text] = 'This is a test-value.'
upvar [utext] = [text]
lowvar [ltext] = [text]
echo [text] # ' in upper-case order: ' # [utext]
echo [text] # ' in lower-case order: ' # [ltext]
end
