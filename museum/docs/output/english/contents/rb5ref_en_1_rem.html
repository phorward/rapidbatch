REM
 
Syntax: REM comment
Description: REM marks a comment-line in the code that is used to comment a scripts. REM does not take any action, it only advises the compiler/interpreter to ignore the current line.
Examples: rem displaying the text "Hello World"
echo 'Hello World'
end
