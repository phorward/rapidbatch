READFILE
 
Syntax: READFILE line = filename, line-to-read
Description: Reads the line at number line-to-read from the file specified at filename, and returns this line. If the file's end is reached or if the file couldn't be found, [errorcode] gets -1 and the return value is "EOF", which means "End Of File".If the assigned line-number is lower than 1, READFILE reads the whole file content and returns it.
Examples: rem read first line...
readfile [text] = 'c:\test.txt', '1'

rem read the whole file...
readfile [text] = 'c:\test.txt', '0'

rem display the first 10 lines from the file
[i] = '0'
[text] = ''
repeat
    [i] + '1'
    readfile [line] = 'C:\jmksf\rb5\readme.txt',[i]
    if [line] ! 'EOF'
        [text] # [line] # [new_line]
    endif
until [i] = '10' | [line] = 'EOF'

echo [text]
Remarks: Use the GETFILE-function, to read data on byte-level from a file (e.g. a binary-file).
