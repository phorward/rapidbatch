[InputBox_Ok], [InputBox_Cancel], [InputBox_X], [InputBox_Y]
 
Description: Using the variables [InputBox_Ok] and [InputBox_Cancel], the captions of the OK- and Cancel-buttons displayed on the build-in INPUTBOX-dialog can be changed and modified.

By setting the variables [InputBox_X] and [InputBox_Y] to any desired pixel value, the dialog's default position on the particular axis can be set and modified. By setting the particular variable to a value of 0, the dialog is automatically centered on the particular axis. This is the default setting.
