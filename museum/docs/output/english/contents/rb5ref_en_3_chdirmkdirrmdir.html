CHDIR, MKDIR, RMDIR
 
Syntax: CHDIR path
MKDIR path
RMDIR path
Description: CHDIR changes the current working directory to the specified path. MKDIR creates the specified path.
The RMDIR-instruction is used to delete a directory, which must be completely empty.
Examples: chdir 'c:\jmksf\rb5'
mkdir 'c:\test'
rmdir 'c:\test'
Remarks: If the particular operation succeeded, [errorcode] gets the value '0', else '-1'.
This makes it possible to check if e.g. a directory could be created or not.
