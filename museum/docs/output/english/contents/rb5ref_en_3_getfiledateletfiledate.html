GETFILEDATE, LETFILEDATE
 
Syntax: GETFILEDATE date-of-last-modification = filename
LETFILEDATE filename, date-of-last-modification
Description: GETFILEDATE reads the date a file was modified on, and returns this in the format "mmddyyyy".
LETFILEDATE resets the modification date of a file to any desired date. date-of-last-modification is in this case specified the same way as returned by GETFILEDATE.
Examples: newfile 'test.txt'
getfiledate [d] = 'test.txt'

echo 'test.txt was modified on ' # [d]

rem Now we reset the date to February 01, 2004
letfiledate 'test.txt', '02012004'

getfiledate [d] = 'test.txt'
echo 'test.txt was "modified" on ' # [d]
delfile 'test.txt'
Remarks: If the file was not found, both LETFILEDATE and GETFILEDATE assign the value '-1' to [errorcode], else '0'. The returned value of GETFILEDATE is in such case an empty-string.
Use GETFILETIME and LETFILETIME to change the last-modification time of a file.
