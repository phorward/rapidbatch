GETPOS
 
Syntax: GETPOS position = expression, substr, count
Description: Finds the specified sub-string (substr-parameter) within the string-expression and returns its character position.
By using the count-parameter it is also possible to determine the as count-parameter specified appearance of substr within the string-expression.

If the string defined as substr was not found within the string-expresson, '0' is returned.
Examples: getpos [pos] = 'Hello World', 'llo', '1'
echo 'Position is: ' # [pos]

getpos [pos] = 'Hello World', 'l', '3'
echo 'Position is: ' # [pos]
Remarks: If the variable [case_sensitivity] is set to '0', GETPOS will ignore upper-/lower-case order within the substr-parameter, so that also equal strings in another case orders are considered.
