[null]
 
Description: The pseudo-variable [null] is used to pass an unused parameter to a procedure or to absorb the return value of a function if it is not used (sometimes this is the case), for example. So in this spirit, [null] is not a variable. If a value is assigned to [null], it will not stored anywhere into the memory, and if [null] is read, it returns an empty string, so [null] can also be used to reset a variable to an empty string.
