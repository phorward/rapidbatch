[RB_Version], [RB_Pro]
 
Description: Contain information on the currently available version of the RapidBATCH environment the script is executed in. [RB_Version] contains the current version release number, e.g. "5.0".

[RB_Pro] contains the value '0' if the version the script is executed with is a Professional Edition of RapidBATCH. If the used version is a Standard Edition, [RB_Pro] is '-1'.
