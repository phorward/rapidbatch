NEWFILE, DELFILE
 
Syntax: NEWFILE filename
DELFILE filename
Description: NEWFILE creates a new, empty file in the file-system. If the file does already exist, it is overwritten by the new file.
DELFILE deletes a file irrepealable from the file system (it is not moved into the Recycle Bin first!).
Examples: rem create new file
newfile 'my_file.txt'

rem delete a temp-file
delfile 'output.tmp'
Remarks: If the particular operation succeeded, [errorcode] gets the value '0', else '-1'.
