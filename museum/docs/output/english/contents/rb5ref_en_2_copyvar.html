COPYVAR
 
Syntax: COPYVAR sub-string = expression, start, length
Description: Extracts a substring from the specified value, that begins at the character on position start and has from this position a lengh of as length specified characters.
The result is the sub-string extracted from the string expression.
Examples: [name] = 'Arnold Schwarzenegger'
rem       123456789012345678901 (character positions)
copyvar [lastname] = [name], '8', '14'
copyvar [firstname] = [name], '1', '6'
echo [lastname] # ', ' # [firstname]
Remarks: If the value '-1' is specified as start parameter value, the length of characters are copied from the right of the string expression.
