ECHO
 
Syntax: ECHO message-text
Description: Displays the expression specified as message-text in a messagebox with single OK-button.
Examples: echo 'Hello World'

[res] = '5' + '4'
echo 'The result is: ' # [res]
Remarks: The default-text for the titlebar of the ECHO-messagebox can be customized over the variable [Echo_Title]. By default, it is "RapidBATCH Echo".
Use the MSGBOX-function, to display an individually designed messagebox-dialog to the user.
