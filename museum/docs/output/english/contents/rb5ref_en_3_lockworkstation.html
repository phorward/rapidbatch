LOCKWORKSTATION
 
Syntax: LOCKWORKSTATION
Description: Locks a Windows 2000, Windows XP, Windows 2003 or Windows Vista Workstation.
If this instruction is called on Windows 95/98/ME/NT, nothing will happen.
Examples: rem lock current workstation?
confirm [lock] = 'Lock computer?'
if [lock] = [true]
    rem lock the computer!
    lockworkstation
endif
Remarks: [errorcode] gets '-1', if RapidBATCH was not able to lock the workstation, else '0'.
