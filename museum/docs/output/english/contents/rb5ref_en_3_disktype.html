DISKTYPE
 
Syntax: DISKTYPE type-id = drive-letter
Description: Gives information about the type a drive has. The return value is an ID with the following meaning:

0 = Unknown medium
1 = removable memory
2 = Fixed medium (e.g. a hard-disk)
3 = Remote drive
4 = CD-ROM drive
5 = RAM-Disk (e.g. USB-Stick)
Examples: disktype [type] = 'C:\'
if [type] = '0' [type] = 'unknwon '
if [type] = '1' [type] = 'removeable-'
if [type] = '2' [type] = 'fixed '
if [type] = '3' [type] = 'remote-'
if [type] = '4' [type] = 'CD-ROM-'
if [type] = '5' [type] = 'RAM-Disk-'

echo 'Drive C: is a ' # [type] # 'medium'
