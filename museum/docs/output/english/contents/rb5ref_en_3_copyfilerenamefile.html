COPYFILE, RENAMEFILE
 
Syntax: COPYFILE source-file, target-file
RENAMEFILE source-file, target-file
Description: COPYFILE copies the as source-file specified file to the as target-file specified file.
RENAMEFILE moves/renames the as source-file specified file to the as target-file specified file.

Both parameters can be specified as relative or absolute file-paths. The target-file can be the path to any other folder or drive.
Examples: rem copy a file...
copyfile 'c:\jmksf\rb5\samples\hello.rb', 'a:\hello.rb'

rem renaming a file...
renamefile 'c:\test.neu', 'c:\test.old'
Remarks: If the particular operation succeeds, [errorcode] get the value '0', else '-1'.
