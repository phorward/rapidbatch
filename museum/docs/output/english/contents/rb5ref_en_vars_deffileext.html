[OpenBox_DefFileExt], [SaveBox_DefFileExt]
 
Description: These two variables allow the definition of a default file extension that is automatically appended to a filename entered in an OPENBOX- resp. SAVEBOX-dialog without any extension. This extension must be assigned without the dot, e.g. ist must be set to 'TXT' for text-files.
