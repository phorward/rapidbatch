GETCHR, GETASC
 
Syntax: GETCHR character = ascii-value
GETASC ascii-value = character
Description: GETCHR returns the particular ASCII-character to the specified ASCII-code value.
GETASC returns the particular ASCII-code value to the specified ASCII-character.
Examples: getchr [zeichen] = '65'
echo 'Character to code 65: ' # [zeichen]
getasc [ascii] = 'B'
echo 'ASCII-Code of character "B" ' # [ascii]
Remarks: none
