[XPStyleActive], [Metric_Menu_Height], [Metric_Caption_Height], [Metric_Edge_Height], [Metric_Edge_Width]
 
Description: The variable [XPStyleActive] contains the value '0' (or [true]), if the operating system the script is executed on is a Windows XP or Windows 2003 system with activated Windows XP Theme. In classic Windows-theme mode or on prior Winwdows-Systems like 2000 or 98, [XPStyleActive] always contains '-1' (or [false]).

The variables [Metric_Menu_Height], [Metric_Caption_Height], [Metric_Edge_Height] and [Metric_Edge_Width] contain informations about the height of a menu-bar, the height of a caption and the height and width of a window's border edge. Use these variables to adjust your customized application windows for best appearance and usability on Windows XP themes and on Windows-systems with customized window appearance settings.
