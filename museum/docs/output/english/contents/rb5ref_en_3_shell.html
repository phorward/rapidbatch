SHELL
 
Syntax: SHELL path, showmode
Description: Runs an external program.

The script calls the program parallel, so that both the script and the program are executed at the same time (multitasking). The instruction expects the program path (with .EXE, .COM or .BAT extension) and a so called show mode, specified over the showmode-parameter. Possible values are 'show' (or '0'), 'hide' (or '1'), 'maximized' (or '2') and 'minimized' (or '3').
Examples: shell 'c:\windows\calc.exe', 'show'
shell 'c:\windows\desktop\test.bat', 'hide'
Remarks: [errorcode] gets a value > 32 if it succeeds. This number is the system process ID of the program.
