CALL
 
Syntax: CALL path, showmode
Description: Runs an external program in modal-mode. That means that the RapidBATCH script suspends its execution until the called program terminates.
Equivalently to the SHELL-instruction, CALL expects as second parameter a show-mode, represented by a number or one of the build-in constants. Possible values are 'show' (or '0'), 'hide' (or '1'), 'maximized' (or '2') and 'minimized' (or '3').
Examples: echo 'Running Notepad...'
call 'notepad.exe','show'
echo 'Notepad was terminated.'
end
Remarks: [errorcode] gets '-1', if an error occurs, else the program's exit code.
