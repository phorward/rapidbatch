GETTOK
 
Syntax: GETTOK token = expression, separation-string, token-counter
Description: The tokenizer-function GETTOK returns a so called token from a string-expression. A token is a sub-string, separated by a special delimiter-character (or delimiter-string) from other tokens. When GETTOK is executed, it returns the token positioned at the as count-parameter specified token-position in the string-expression.
Examples: findfiles [files] = 'c:\jmksf\rb5\*.exe', '1'

[count] = '1'
repeat
    gettok [tok] = [files], '|', [count]
    if [tok] ! ''
        echo 'Token at position ' # [count] # ' is: ' # [tok]
    endif
    [count] + '1'
until [tok] = ''
Remarks: If the end of the string is reached, GETTOK returns all characters that had been read until the string-end from the last token-delimiter.
If the variable [case_sensitivity] is set to '0', GETTOK will also consider token-separators which don't fit the case order of the specified token-delimiter string.
