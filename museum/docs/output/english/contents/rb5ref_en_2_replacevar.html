REPLACEVAR
 
Syntax: REPLACEVAR result = expression, find, replace
Description: Replaces every appearance of the specified search-string find within the string-expression and replaces it by the string replace.
Examples: chdir 'c:\jmksf\rb5'
findfiles [files] = '*.exe', '1'
replacevar [files] = [files], '|', [new_line]
echo 'Content of ' # [current] # [new_line] # [new_line] # [files]
Remarks: If the variable [case_sensitivity] has the value '0', REPLACEVAR will also replace strings that do not fit the same case order as specified as find-parameter.
