TRIMVAR
 
Syntax: TRIMVAR result = expression
Description: Removes all blanks from in front of and behind a string-expression.
Examples: [text] = '    RapidBATCH  '
echo '***' # [text] # '***'
trimvar [trim_text] = [text]
echo '***' # [trim_text] # '***'
end
