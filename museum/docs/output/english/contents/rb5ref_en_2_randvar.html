RANDVAR
 
Syntax: RANDVAR random-value = range
Description: Generates a pseudo-random-number between 0 and the specified value range - 1.
Examples: rem Generate random value...
randvar [random] = '255'
echo 'Random-value: ' # [random]
Remarks: At the implementation of RANDVAR, the problem was noticed that the same set of random numbers sometimes is returned continual. This problem has been solved as far as possible, but it may happen that a muster of numbers is always re-generated. RANDVAR uses the system time as base for its numbers.
