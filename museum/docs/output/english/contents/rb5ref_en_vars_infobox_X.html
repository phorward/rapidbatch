[InfoBox_Enabled], [InfoBox_Height], [InfoBox_Width], [InfoBox_X], [InfoBox_Y]
 
Description: [InfoBox_Enabled] lets you toggle the visibility mode of the INFOBOX build-in dialog. By setting [InfoBox_Enabled] to '0' or 'show', the dialog is show, by setting it to '-1' or 'hide', it is hidden from the screen.

By setting the variables [InfoBox_X] and [InfoBox_Y] to any desired pixel value, the dialog's default position on the particular axis can be set and modified. By setting the particular variable to a value of 0, the dialog is automatically centered on the particular axis. This is the default setting.

The variables [InfoBox_Width] and [InfoBox_Height] let you change and modify the dialog's default size.
