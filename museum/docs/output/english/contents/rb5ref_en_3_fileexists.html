FILEEXISTS
 
Syntax: FILEEXISTS result = filename
Description: Verifies a file's existence. If the as filename specified file was found, the function returns '0', else '-1'.
Examples: fileexists [errorcode] = 'c:\windows\calc.exe'
if [errorcode] = '0' echo 'The program file CALC.EXE exists!'
Remarks: none
